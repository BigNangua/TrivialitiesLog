### 访问github

通过修改hosts文件访问github（对浏览器能访问官网，但是不能拉取项目也有效果）

打开：DNS解析

```
http://www.ip33.com/dns.html
```

![20250122104317](.\img\20250122104317.png)

在响应IP中选择一个能够ping通的ip

修改hosts

```
C:\Windows\System32\drivers\etc\hosts
```

添加：

```
[响应IP] github.com
```
